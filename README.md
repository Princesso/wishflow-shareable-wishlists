# Wishflow - Shareable Wishlists

Wishflow is a modern, buildless web application that allows users to create and share beautiful wishlists. 

## Features
- **Create Lists**: Add items with links, prices, and notes.
- **Cash Funds**: Add Venmo/PayPal details for cash gifts.
- **AI Assistant**: Get gift suggestions based on interests (powered by on-device WebLLM).
- **Share**: Generate unique links to share with friends.
- **Guest View**: Friends can view lists and locally mark items as "Purchased".

## Tech Stack
- HTML5, CSS3, JavaScript (jQuery)
- Tailwind CSS (via CDN)
- WebLLM (for local AI)
- No Backend (Local Storage & URL encoding)
